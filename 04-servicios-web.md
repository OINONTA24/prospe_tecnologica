# Servicio Web

En esta sección se presentan las prácticas realizadas para comprender cómo se crean y diseñan los servicios web, así como para conocer su arquitectura. Se enfoca en su implementación, su programación, el envío de instrucciones desde un sistema embebido y el desarrollo e integración del frontend y backend.
